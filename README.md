# Privacy Policy - Spot the Canadian Chrome Extension

**Last Updated:** February 2026

## Information We Collect

### Analytics Data
We use PostHog analytics to understand how our extension is used and improve your experience. We collect:

- **Anonymous Usage Data:** Extension installation, daily active users, celebrity spotting events
- **Performance Metrics:** Data source usage (Pantheon API vs local), scan timing
- **Non-Personal Information:** No personal identifiers, IP addresses, or sensitive data

### Local Storage
- **Unique User ID:** Generated randomly for analytics purposes only
- **Installation Status:** Tracks if extension has been installed before
- **No Personal Data:** We never store names, emails, or personal information

## How We Use Your Data

### Analytics Only
- **Improve Performance:** Monitor extension speed and reliability
- **User Insights:** Understand usage patterns without identifying individuals
- **Feature Development:** Make data-driven decisions about improvements

### No Third-Party Sharing
- **PostHog Analytics:** Data sent to PostHog for analysis only
- **No Selling:** We never sell, rent, or share your data with third parties
- **Anonymous Only:** All analytics data is anonymized

## Data Security

### Protection Measures
- **Local Storage:** Data stored securely in Chrome's local storage
- **Encrypted Transmission:** Analytics data sent over HTTPS
- **Minimal Collection:** We only collect what's necessary for improvement

### Data Retention
- **Analytics Data:** Retained according to PostHog's standard policies
- **Local Data:** Stored only on your device, can be cleared anytime

## Your Rights

### Control Your Data
- **Clear Data:** Remove local storage anytime via Chrome settings
- **Opt Out:** Disable extension by removing it from Chrome
- **Transparency:** Full visibility into what we collect

### Contact Us
Questions about privacy? Contact us through the Chrome Web Store extension page.

## Changes to This Policy

We may update this privacy policy as our extension evolves. Changes will be reflected in the Chrome Web Store listing and within the extension itself.

---

**This privacy policy applies only to the "Spot the Canadian" Chrome extension and not to any websites or services you visit while using our extension.**
